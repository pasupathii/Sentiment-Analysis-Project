# Sentiment Analysis Using NLTK

## Live Demo

[Click here to view the hosted project](https://sentiment-analysis-project-4qfqjao7yuzpoq9otsbwff.streamlit.app/)

## Project Overview :

This project focuses on performing sentiment analysis using Natural Language Processing and Machine Learning techniques. The system analyzes textual data and classifies the sentiment of the text as positive, negative, or neutral based on extracted features.

The project uses NLTK for text preprocessing, Random Forest for classification, clustering techniques for grouping similar text patterns, and various machine learning models for performance comparison.

## Features :

Text preprocessing using NLTK
Removal of stop words, punctuation, and special characters
Tokenization and stemming/lemmatization
Feature extraction using TF-IDF or Count Vectorizer
Sentiment classification using Random Forest Algorithm

 Technologies Used:

  - Python
  - NLTK
  - Scikit-learn
  - Pandas
  - NumPy


## Dataset:

The DATASET: contains text reviews, comments, or feedback along with their sentiment labels. The dataset is used to train and test the machine learning models for sentiment prediction.

Example sentiment classes:

 - Positive
 - Negative
 - Neutral

## Installation

Clone the repository:

git clone https://github.com/pasupathii/sentiment-analysis-using-nltk.git

Go to the project folder:

cd sentiment-analysis-using-nltk

Install the required packages:

pip install -r requirements.txt
Required Libraries
pip install nltk pandas numpy scikit-learn matplotlib
How to Run the Project

Run the Python file:

python sentiment_analysis.py

Or run the Jupyter Notebook:

jupyter notebook

Then open the project notebook file and execute the cells step by step.
